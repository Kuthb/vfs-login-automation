

# VFS Global Login Automation — Selenium

Selenium 4 + Python-based automation for the VFS Global UAE–Malta login portal.  
Supports both headed and headless Chrome execution with automatic fallback, structured logging, screenshot capture, and flexible credential management.

This project demonstrates real-world browser automation with handling for dynamic UI elements, cookie consent, and bot protection mechanisms.

---

## 🚀 Tech Stack

| Tool | Purpose |
|------|--------|
| Selenium 4 | Browser automation |
| undetected-chromedriver | Helps bypass basic bot detection |
| python-dotenv | Loads environment variables |
| Python 3.8+ | Runtime |
